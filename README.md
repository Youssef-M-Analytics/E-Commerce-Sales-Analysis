## 📌 Project Overview
This project focuses on analyzing an **E-commerce dataset** to transform raw transaction data into actionable business insights. I built a **Dynamic & Interactive Dashboard** from scratch using Microsoft Excel, covering the entire data analysis lifecycle: data cleaning, processing, and visualization.

## 🎯 Business Objectives
The dashboard was designed to answer critical business questions:
* Which product categories are driving the most **Profit**?
* How is the **Sales Distribution** spread across different states and regions?
* What is the **Year-over-Year (YoY) Growth** for key metrics?
* Identification of the **Top 5 Sub-Categories** by sales volume.

## 🛠️ Technical Toolkit & Skills
* **Advanced Excel:** Data cleaning, handling missing values, and formatting.
* **Pivot Tables & Charts:** Summarizing thousands of rows for efficient analysis.
* **Custom Calculations:** Implementing logic for **Profit Margin %** and **YoY Growth**.
* **Dynamic Visualization:** * **Combo Charts:** Visualizing Sales and Profit trends simultaneously.
    * **Waterfall Charts:** Analyzing profit contribution by category.
    * **Map Charts:** Displaying geographical performance.
* **Interactivity:** Using **Slicers & Timelines** to create a fully interactive user experience.

## 📊 Key Insights & KPIs
The dashboard tracks the following core metrics:
* **Total Sales & Profit:** Overall financial health.
* **Quantity Sold:** Inventory movement analysis.
* **Category Share:** Visualizing which categories dominate the market.
* **Regional Performance:** Identifying high-performing vs. low-performing areas.

## 📂 Project Structure
* `Data/`: Contains the raw e-commerce dataset used for analysis.
* `Dashboard/`: Contains the final interactive Excel Dashboard (.xlsx).
